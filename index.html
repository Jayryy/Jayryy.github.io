Activities

Managing corporate servers with a DevOps culture is the main topic of this course. Students will learn how to use Ansible for configuration management, the standard monitoring practices, and Git for collaboration in order to run a DevOps environment on Linux servers. They will also learn how to use cloud services and containerized technologies to create an immutable environment and monitor and leverage alerting in the DevOps environment. This course is also intended to reflect the problems that we currently face in the actual world and how to resolve them.

Learnings

The management of corporate servers within a DevOps culture is the main topic of CPE232. Learn how to monitor and use alerts in a DevOps environment by using Ansible for configuration management and de facto standard monitoring processes in this course. Additionally, you will become proficient in the principles of managing a DevOps environment on Linux servers.
